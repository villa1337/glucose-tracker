# Control de Glucosa

Aplicación web para el seguimiento de niveles de glucosa y medicamentos.

## Características

- **Control de Glucosa**: Registro y seguimiento de mediciones
- **Historial**: Gráficos y tablas de mediciones anteriores
- **Medicamentos**: Horarios y recordatorios de medicación
- **Interfaz en Español**: Diseñada para adultos mayores

## Desarrollo

```bash
npm install
npm run dev
```

## Despliegue en Vercel

```bash
npm install -g vercel
vercel
```

## Uso

1. **Registrar Medición**: Ingresa el valor de glucosa y selecciona si es en ayunas o después de comer
2. **Ver Historial**: Revisa gráficos y tablas de mediciones anteriores
3. **Medicamentos**: Consulta horarios de medicación

Los datos se guardan localmente en el navegador.
